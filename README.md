# GenMind AI 🤖

GenMind AI is a local generative AI assistant built using Streamlit.

## Features
- ChatGPT-style interface
- Streaming responses
- Local AI models
- Clean UI

## Tech Stack
Python  
Streamlit  
Ollama

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
